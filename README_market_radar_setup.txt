Market Radar 무료서버 연결 순서

1) Cloudflare 계정 생성/로그인
2) Workers & Pages 메뉴 진입
3) Create application 또는 Create Worker 선택
4) Worker 이름 아무거나 입력: market-radar-proxy
5) 코드 편집 화면에서 market-radar-worker.js 내용을 전체 붙여넣고 Deploy
6) 발급된 주소 복사
   예: https://market-radar-proxy.<계정명>.workers.dev
7) market_radar_dashboard_yahoo_v5_worker.html 열기
8) 우측 상단 '서버주소' 버튼 클릭
9) 복사한 Worker 주소 붙여넣기
10) 새로고침 버튼 클릭

성공하면 업데이트 문구에 '내 무료서버 연결'이 표시됩니다.
